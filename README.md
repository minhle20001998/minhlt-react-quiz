# Web Game
Quiz game website using ReactJS, Typescript, Context, custom Hooks

# Author: MinhLT 

# Live Demo:
https://minhlt-quiz-game.netlify.app/

## How to run on local env
1. Install dependencies
```
npm install
```
2. Serve on local
```
npm start
```
3. Open website and direct to URL
```
http://localhost:3000/
```
