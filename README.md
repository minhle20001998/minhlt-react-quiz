Quiz game website using ReactJS, Typescript, Context, custom Hooks

# Live Demo:

## How to run on local env
1. Install dependencies
```
npm install
```
2. Serve on local
```
npm start
```
3. Open website and direct to URL
```
http://localhost:3000/
```
